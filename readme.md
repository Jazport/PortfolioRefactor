# Jazlyn Portillo Portfolio

Welcome to my personal portfolio website! This portfolio showcases my projects, reflections, resume, and provides a way to contact me. It was built using **HTML, CSS, and JavaScript** and is fully responsive across devices.

---

## Features

- **Responsive Design:** Works seamlessly on desktop, tablet, and mobile.
- **Interactive Hero Section:** Animated background and smooth scroll buttons.
- **About/Bio Section:** Clean layout with hover effects on the bio card.
- **Projects & Reflections:** 3D-like cards with hover animations and overlay descriptions.
- **Resume Section:** Downloadable PDF with interactive preview.
- **Contact Form:** Validates entries and displays success/error messages with animation.
- **Navigation:** Mobile-friendly toggle menu for easy access to pages.
- **Footer:** Social links with subtle hover animations.

---

## Technologies Used

- **HTML5**
- **CSS3** (including Flexbox, Grid, Animations)
- **JavaScript** (form validation, smooth scroll, animations)
- **Google Fonts:** [Dancing Script](https://fonts.google.com/specimen/Dancing+Script) for logo

---

## How to Run

You can view the portfolio **hosted live** at: [https://refactoredportfolio.netlify.app/](https://refactoredportfolio.netlify.app/)

Or, to run locally:

1. Clone or download this repository.
2. Open `index.html` in your web browser.
3. Navigate through the pages to view projects, resume, reflections, and contact form.

---

## Notes

- The contact form currently performs **front-end validation only**.
- All links and buttons are interactive and styled with hover animations for a modern feel.
- The portfolio is designed with **creative gradients, 3D card effects, and smooth transitions** to enhance the user experience.
